## 🎯 게임 구성 요소

### 🎲 주사위

| 항목          | 내용                                           |
| ------------- | ---------------------------------------------- |
| **종류**      | 6면 주사위 (1~6)                               |
| **용도**      | 점수 획득, 특수 효과 발동                      |
| **특수 효과** | 주사위 6(공격), 2(방어) 시 랜덤 특수 효과 발동 |

### 🔋 에너지

| 항목            | 내용                              |
| --------------- | --------------------------------- |
| **획득**        | 매 라운드 시작 시 2~3점 랜덤 지급 |
| **최대치**      | 10점                              |
| **용도**        | 카드 사용, 방어막 생성            |
| **승리 보너스** | 라운드 승리 시 +1점 추가          |

### ⚔️ 성향 시스템

| 성향       | 특징                 | 성공 조건       | 보너스               |
| ---------- | -------------------- | --------------- | -------------------- |
| **공격**   | 높은 점수 노리기     | 주사위 4~6      | 주사위 6 시 +3점     |
| **방어**   | 안정적 점수 + 방어막 | 주사위 2~5      | 성공 시 방어막 +1    |
| **밸런스** | 균형잡힌 성장        | 주사위 1~6 전부 | 에너지 +1, 게이지 +1 |

### 📊 카드 게이지 시스템

| 항목          | 내용                                                          |
| ------------- | ------------------------------------------------------------- |
| **목적**      | 전략 카드 획득을 위한 누적 포인트 시스템                      |
| **획득 조건** | 같은 눈금 2회 이상, 연속 6 두 번, 누적 점수 15점, 2연패/3연승 |
| **최대값**    | 5 (도달 시 카드 지급 후 0으로 초기화)                         |

### 🃏 전략 카드

| 항목          | 내용                                                         |
| ------------- | ------------------------------------------------------------ |
| **획득**      | 카드 게이지 5 도달 시 1장 지급                               |
| **종류**      | 공격, 방어, 부스트, 스틸, 리셋                               |
| **덱 구성**   | 공격 7장, 방어 7장, 부스트 3장, 스틸 2장, 리셋 1장 (총 20장) |
| **덱 관리**   | 덱 소진 시 전체 셔플 후 재사용 (확률 유지)                   |
| **뽑기 확률** | 공격/방어 각 35%, 부스트 15%, 스틸 10%, 리셋 5%              |

### 🛡️ 방어막

| 항목          | 내용                                           |
| ------------- | ---------------------------------------------- |
| **생성**      | 에너지 3점으로 방어막 1개 생성                 |
| **최대 보유** | 2개                                            |
| **효과**      | 상대의 직접적 공격 효과 차단                   |
| **자동 생성** | 방어 성향 성공 시 방어막 +1 (에너지 소모 없음) |

### 🎭 이벤트 카드 시스템

| 항목     | 내용                                          |
| -------- | --------------------------------------------- |
| **획득** | 매 라운드 종료 후 플레이어와 AI 각각 1장 뽑기 |
| **종류** | 긍정적(60%), 부정적(20%), 중립적(20%) 이벤트  |
| **효과** | 즉시 적용 또는 지연 적용 (다음 라운드)        |
| **차단** | 방어막으로 일부 이벤트 효과 차단 가능         |

### 📈 점수 시스템

| 항목            | 내용                                          |
| --------------- | --------------------------------------------- |
| **구조**        | 라운드 점수(임시) + 총점(누적) 이중 구조      |
| **라운드 점수** | 매 라운드 초기화되는 임시 점수                |
| **총점**        | 게임 전체 누적되는 최종 점수 (승부 결정 기준) |
| **최솟값 보장** | 라운드 점수 0점, 총점 1점                     |

### 🏆 승리 조건

| 항목          | 내용                                                       |
| ------------- | ---------------------------------------------------------- |
| **기본 조건** | 최종 라운드 종료 후 총점이 더 높은 쪽이 승리               |
| **동점 처리** | 마지막 라운드 점수 → 에너지 보유량 → 소유 카드 수 → 무승부 |

## ⚔️ 게임 진행

### 🔄 라운드 구조

#### 1️⃣ **라운드 시작 시 전략적 의사결정**

- 성향 선언 (공격/방어/밸런스)
- 카드 사용 계획 (이번 라운드 vs 다음 라운드)
- 상대 상황 분석 (점수, 에너지, 카드 상황 파악)

#### 2️⃣ **턴 진행**

- **라운드 시작 전**: 방어 카드, 부스트 카드 사용 (선택사항)
- 플레이어와 AI가 주사위 굴림
- **주사위 굴림 직후**: 공격 카드, 스틸 카드, 리셋 카드 사용 (선택사항)
- 점수 합산 및 특수 효과 발동
- 방어막 판정

#### 3️⃣ **라운드 종료**

- 랜덤 이벤트 발생
- 점수 비교 및 누적 합산

## 🎯 게임 단계별 행동 가이드

### 📋 라운드 진행 순서

| 단계   | 시점             | 행동               | 선택사항                                         |
| ------ | ---------------- | ------------------ | ------------------------------------------------ |
| **1️⃣** | 라운드 시작 전   | **성향 선언**      | 공격/방어/밸런스 중 선택                         |
|        |                  | **카드 사용 계획** | 이번 라운드 vs 다음 라운드 사용 결정             |
|        |                  | **상대 상황 분석** | 점수, 에너지, 카드 상황 파악                     |
| **2️⃣** | 라운드 시작 시   | **방어 카드**      | 다음 공격에 대한 방어 준비 (에너지 1 소모)       |
|        |                  | **부스트 카드**    | 이번 라운드 점수 2배 효과 준비 (에너지 4 소모)   |
| **3️⃣** | 주사위 굴림      | **주사위 굴림**    | 플레이어와 AI가 각각 6면 주사위 1개 굴림         |
|        |                  | **점수 계산**      | 성향별 점수 계산 및 특수 효과 발동               |
| **4️⃣** | 주사위 굴림 직후 | **공격 카드**      | 상대 점수 감소 (에너지 2 소모)                   |
|        |                  | **스틸 카드**      | 상대 에너지 1~2점 훔치기 (에너지 2 소모)         |
|        |                  | **리셋 카드**      | 이번 라운드 점수 초기화 (에너지 4 소모)          |
| **5️⃣** | 점수 계산        | **효과 적용**      | 기본 점수 + 상황 보너스 + 특수 주사위 효과       |
|        |                  | **카드 효과**      | 전략 카드 효과 적용                              |
|        |                  | **방어막 판정**    | 공격 차단 (방어막 1개당 공격 1회 차단)           |
| **6️⃣** | 이벤트 처리      | **이벤트 카드**    | 플레이어와 AI 각각 1장 뽑기                      |
|        |                  | **즉시 효과**      | 폭탄, 도둑, 점수교환 등 즉시 적용                |
|        |                  | **지연 효과**      | 행운의 여신, 저주, 보너스 턴 등 다음 라운드 예약 |
| **7️⃣** | 라운드 종료      | **점수 누적**      | 라운드 점수를 총점에 누적                        |
|        |                  | **초기화**         | 라운드 점수 초기화, 다음 라운드 준비             |

### ⚡ 성향별 행동 전략

| 성향       | 🗡️ 공격                                        | 🛡️ 방어                                         | ⚖️ 밸런스                          |
| ---------- | ---------------------------------------------- | ----------------------------------------------- | ---------------------------------- |
| **목표**   | 높은 점수로 압도적 승리                        | 안정적 점수 + 방어막 확보                       | 균형잡힌 성장과 자원 확보          |
| **핵심**   | 주사위 4~6 성공 시 높은 점수, 6일 때 특수 효과 | 주사위 2~5 성공 시 점수 획득, 성공 시 방어막 +1 | 모든 주사위 결과가 점수, 실패 없음 |
| **보너스** | 주사위 6 시 +3점 추가                          | 성공 시 방어막 무료 생성                        | 항상 에너지 +1, 카드 게이지 +1     |
| **성공률** | 50% (4~6)                                      | 67% (2~5)                                       | 100% (1~6)                         |
| **위험도** | 높음 (실패 시 1점)                             | 중간 (안정적)                                   | 낮음 (항상 성공)                   |

### 🎲 특수 주사위 효과 활용

#### ⚔️ 공격 주사위 효과 (공격 선언 + 주사위 6 시)

| 효과     | 아이콘 | 설명                      | 방어막 차단                  |
| -------- | ------ | ------------------------- | ---------------------------- |
| **공격** | ⚡     | 상대 점수 -2              | ✅ 가능                      |
| **화염** | 🔥     | 상대 점수 -3 + 다음 턴 -1 | ⚠️ 부분 차단 (이번 라운드만) |
| **더블** | 🎯     | 자신 점수 2배             | ❌ 불가 (자신에게 유리)      |

#### 🛡️ 방어 주사위 효과 (방어 선언 + 주사위 2 시 랜덤 발동)

| 효과              | 아이콘 | 설명                          | 방어막 차단 | 특이사항       |
| ----------------- | ------ | ----------------------------- | ----------- | -------------- |
| **방어 주사위**   | 🛡️     | 즉시 일회성 방어막 1개 생성   | ❌ 불가     | 자원 생성 효과 |
| **보너스 주사위** | ⭐     | 자신 이번 라운드 점수 + (2~6) | ❌ 불가     | 자신에게 유리  |

#### 🛡️ 방어막과의 상호작용

| 효과          | 차단 가능 여부 | 차단 범위              | 특이사항                |
| ------------- | -------------- | ---------------------- | ----------------------- |
| **⚡ 공격**   | ✅ 가능        | 상대 점수 -2 완전 차단 | 방어막 1개 소모         |
| **🔥 화염**   | ⚠️ 부분 차단   | 이번 라운드 -3만 차단  | 다음 턴 -1은 차단 불가  |
| **⭐ 보너스** | ❌ 불가        | 보너스 점수            | 자신에게 유리한 효과    |
| **🎯 더블**   | ❌ 불가        | 점수 2배 효과          | 자신에게 유리한 효과    |
| **지연 효과** | ❌ 불가        | 다음 턴 효과           | 시간차 효과는 차단 불가 |

| 항목          | 내용                                      |

| **생성 조건** | 에너지 3점으로 방어막 1개 생성            |
| **최대 보유** | 2개까지                                   |
| **소모 방식** | 공격을 받을 때마다 방어막 1개씩 자동 소모 |

#### ✅ 차단 가능한 효과

| 효과               | 설명                          | 차단 방식       |

| **🔥 화염 주사위** | 상대 점수 -3 (이번 라운드만)  | 방어막 1개 소모 |
| **🦹 스틸 카드**   | 에너지 훔치기                 | 방어막 1개 소모 |
| **방어 카드**      | 🛡️ 다음 공격에 대한 방어 효과 | 방어막 1개 소모 |

| -------------------- | ------ | ---------------------------- | -------------------- |
| **⭐ 보너스 주사위** | ⭐     | 자신에게 유리한 효과         | 자신에게 유리한 효과 |
| **🎯 더블 주사위**   | 🎯     | 자신 점수 2배                | 자신에게 유리한 효과 |

| **지연 적용** | 행운의 여신, 저주, 보너스 턴                                   | 🍀🌙🎲       | 다음 라운드 적용 |

> **💡 팁**: 이벤트 효과는 점수 계산의 마지막 단계에서 적용되므로, 다른 모든 효과를 고려한 후 반영됩니다!

## 📊 점수 시스템

### 🎯 점수 계산 체계 개요

**주사위 배틀**의 점수 시스템은 **라운드 점수**와 **총점**의 이중 구조로 운영됩니다.

| 점수 유형       | 설명                             | 특징             |
| --------------- | -------------------------------- | ---------------- |
| **라운드 점수** | 각 라운드에서 획득하는 임시 점수 | 매 라운드 초기화 |
| **총점**        | 게임 전체에서 누적되는 최종 점수 | 승부 결정 기준   |

### 📊 점수 계산 시스템 (개발자/규칙 해석용)

#### 🔄 핵심 계산 순서

| 순서  | 단계        | 설명                                                                          | 세부사항    |
| ----- | ----------- | ----------------------------------------------------------------------------- | ----------- |
| **1** | 기본 점수   | 주사위 결과 + 성향별 성공/실패 판정                                           | 기본 규칙   |
| **2** | 상황 보너스 | 성향별 특별 보너스 (공격+6=+3점, 방어성공=방어막+1, 밸런스=에너지+1/게이지+1) | 성향 보너스 |
| **3** | 특수 주사위 | 랜덤 효과 발동 (⚡공격, 🔥화염, 🎯더블, 🛡️방어, 🚫차단, ⭐보너스)             | 랜덤 효과   |
| **4** | 전략 카드   | 사용한 카드 효과 적용                                                         | 카드 효과   |
| **5** | 방어막 판정 | 공격 차단 (방어막 1개당 공격 1회 차단)                                        | 방어 효과   |
| **6** | 이벤트 효과 | 랜덤 이벤트 적용 (즉시/지연 효과 구분)                                        | 이벤트 효과 |
| **7** | 최종 합산   | 라운드 점수 → 총점 반영                                                       | 최종 점수   |

#### 🔗 주요 상호작용 규칙

| 규칙 유형       | 설명                                                 | 예시        | 세부사항       |
| --------------- | ---------------------------------------------------- | ----------- | -------------- |
| **곱셈 효과**   | 🎯더블(×2) + 부스트카드(×2) = 최종 ×4                | 순차 적용   | 중첩 가능      |
| **감점 누적**   | ⚡공격(-2) + 공격카드(-3) = 총 -5점                  | 단순 합산   | 누적 가능      |
| **방어막 차단** | 공격카드, ⚡공격, 🔥화염(일부), 스틸카드만 차단 가능 | 선택적 차단 | 차단 범위 제한 |
| **최솟값 보장** | 라운드 점수 0점, 총점 1점 (특수 이벤트 제외)         | 안전장치    | 음수 방지      |

### 📝 계산 예시 (핵심 케이스)

#### ⚔️ 공격 + 부스트 조합

| 항목         | 내용                                              | 계산 과정                  | 결과               |
| ------------ | ------------------------------------------------- | -------------------------- | ------------------ |
| **상황**     | 공격 선언 + 주사위 6 + ⚡공격 + 부스트 카드       | 복합 효과                  | 고점수 + 상대 피해 |
| **계산**     | (6 + 3) × 2 = 18점, 상대 -2점                     | 기본점수 + 보너스 × 부스트 | 18점 획득          |
| **특이사항** | 공격 6의 +3점 보너스와 부스트 카드의 ×2 효과 중첩 | 시너지 효과                | 최대 효과          |

#### 🛡️ 방어막 활용

| 항목         | 내용                                                | 계산 과정       | 결과           |
| ------------ | --------------------------------------------------- | --------------- | -------------- |
| **상황**     | 상대 🔥화염 + 공격카드, 내 방어막 2개               | 방어막 vs 공격  | 완벽한 방어    |
| **결과**     | 이번 라운드 피해 없음, 다음 라운드 -1점, 방어막 0개 | 방어막 2개 소모 | 부분 방어 성공 |
| **특이사항** | 🔥화염의 다음 라운드 -1점은 차단 불가               | 지연 효과       | 방어막 한계    |

#### 🧹 리셋 우선순위

| 항목         | 내용                                    | 계산 과정      | 결과          |
| ------------ | --------------------------------------- | -------------- | ------------- |
| **상황**     | 리셋 카드 + 🧹리셋 이벤트               | 카드 vs 이벤트 | 우선순위 결정 |
| **결과**     | 카드 효과가 먼저 적용되어 이벤트 무의미 | 카드 우선      | 이벤트 무효화 |
| **특이사항** | 리셋 카드는 이벤트보다 우선 적용        | 우선순위 규칙  | 전략적 활용   |

> **💡 팁**: 복합 효과를 잘 활용하면 예상보다 높은 점수를 얻을 수 있습니다!
