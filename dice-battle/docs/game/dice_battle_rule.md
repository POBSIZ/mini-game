## 🎮 게임 개요

### 기본 정보

- **플레이어 수**: 1명 (사람) vs AI 상대
- **게임 시간**: 모드별 5~10분
- **목표**: 최종 라운드 종료 후 총점이 더 높은 쪽이 승리

### 게임 모드

- **빠른 모드**: 5라운드 (약 5분 플레이)
- **정규 모드**: 8라운드 (약 8~10분 플레이)
- **확장 모드**: 10라운드 (전략 요소 풍부, 장기전)

---

## 🎯 게임 구성 요소

### 주사위

- **종류**: 6면 주사위 (1~6)
- **용도**: 점수 획득, 특수 효과 발동

### 에너지

- **획득**: 매 라운드 시작 시 2~3점 랜덤 지급
- **최대치**: 10점
- **용도**: 카드 사용, 방어막 생성, 특수 효과 발동

### 전략 카드

- **획득**: 카드 게이지 5 도달 시 1장 지급
- **종류**: 공격, 방어, 부스트, 스틸, 리셋
- **카드 덱 구성**: 공격 7장, 방어 7장, 부스트 3장, 스틸 2장, 리셋 1장 (총 20장)
- **뽑기 확률**: 공격/방어 각 35%, 부스트 15%, 스틸 10%, 리셋 5%

### 방어막

- **생성**: 에너지 품질에 따라 생성 (상세 규칙은 방어막 시스템 참조)
- **최대 보유**: 2개
- **효과**: 상대의 직접적 공격 효과 차단

## ⚔️ 게임 진행

### 라운드 구조

1. **라운드 시작 시 전략적 의사결정**

   - 성향 선언 (공격/방어/밸런스)
   - 에너지 할당 (일반/고급/프리미엄으로 분배)
   - 카드 사용 계획 (이번 라운드 vs 다음 라운드)
   - 상대 상황 분석 (점수, 에너지, 카드 상황 파악)

2. **턴 진행**

   - 플레이어와 AI가 주사위 굴림
   - 점수 합산 및 특수 효과 발동
   - 전략 카드 사용
   - 방어막 판정

3. **라운드 종료**
   - 랜덤 이벤트 발생
   - 점수 비교 및 누적 합산

## 🎲 주사위 시스템

### 성향 선언

매 라운드 시작 시 플레이어는 다음 중 하나를 선택하여 선언합니다:

#### 공격 선언

- **성공**: 주사위 4~6 (눈금 점수 반영)
- **실패**: 주사위 1~3 (점수 1만 반영)
- **특수 효과**: 6일 경우 공격 주사위 효과 랜덤 발동 (⚡공격/🔥화염/🎯더블)
- **상황 보너스**: 성공 시에만 자신의 이번 라운드 기본 점수(주사위 결과)가 6일 경우 +3점 추가 (High Risk, High Return)

#### 방어 선언

- **성공**: 주사위 2~5 (눈금 점수 반영)
- **실패**: 주사위 1 또는 6 (점수 0)
- **특수 효과**: 2일 경우 방어 주사위 효과 랜덤 발동 (🛡️방어/🚫차단/⭐보너스)
- **상황 보너스**: 성공 시에만 방어막 생성 +1개 (에너지 소모 없음)

#### 밸런스 선언

- **성공**: 주사위 1~6 전부 (눈금 그대로 점수)
- **특수 효과**: 없음
- **상황 보너스**: 항상 적용 - 에너지 +1, 카드 게이지 +1 (균형적 성장)

### 특수 주사위 효과 상세

#### 공격 주사위 효과 (주사위 6 시 랜덤 발동)

- ⚡ **공격 주사위**: 상대 이번 라운드 점수 -2 (방어막으로 차단 가능)
- 🔥 **화염 주사위**: 상대 이번 라운드 점수 -3, 다음 턴 시작 시 -1 추가
- 🎯 **더블 주사위**: 자신 이번 턴 점수 2배

#### 방어 주사위 효과 (주사위 2 시 랜덤 발동)

- 🛡️ **방어 주사위**: 상대의 직접 공격 효과 1회 즉시 무효화
- 🚫 **차단 주사위**: 이번 라운드의 모든 특수 효과를 무효화 (상세 범위 아래 참조)
- ⭐ **보너스 주사위**: 자신 이번 라운드 점수 + (2~6)

#### 방어막과의 상호작용

- **차단 가능**: ⚡ 공격, 🔥 화염(해당 라운드 감점 부분만)
- **차단 불가**: 🚫 차단, ⭐ 보너스, 🎯 더블, 🔥 화염의 다음 턴 -1, 모든 지연 효과

#### 🚫 차단 주사위 무효화 범위

**무효화 대상** (이번 라운드에 발생한 모든 특수 효과):

- ⚡ 공격 주사위 (상대 점수 -2 효과 취소)
- 🔥 화염 주사위 (상대 점수 -3 효과 취소, 단 다음 턴 -1은 유지)
- 🎯 더블 주사위 (점수 2배 효과 취소)
- ⭐ 보너스 주사위 (보너스 점수 +2~6 효과 취소)
- 🛡️ 방어 주사위 (방어 효과 취소)
- 전략 카드의 특수 효과 (부스트, 공격, 방어 카드 등)

**무효화 불가**:

- 🔥 화염 주사위의 다음 턴 -1 효과 (이미 예약된 지연 효과)
- 랜덤 이벤트 효과 (시스템 레벨 효과)
- 기본 주사위 점수 (특수 효과가 아닌 기본 점수)
- 상황 보너스 (성향 선언에 따른 보너스)
- **방어막 생성** (에너지로 생성하거나 방어 보너스로 생성된 모든 방어막)
- **기존 보유 방어막** (이미 보유 중인 방어막의 차단 효과)

**발동 타이밍**: 라운드 종료 직전, 모든 효과 적용 후 일괄 무효화

## 🔋 에너지 시스템

### 에너지 품질 시스템

- **일반 에너지 (1~3)**: 기본 효과
- **고급 에너지 (4~6)**: 효과 2배
- **프리미엄 에너지 (7~10)**: 효과 3배

### 에너지 관리

- **획득**: 매 라운드 시작 시 2~3점 랜덤 지급
- **보장**: 직전 라운드 패배 시 최소 3점 이상
- **승리 보너스**: 라운드 승리 시 +1점 추가
- **과부하**: 에너지 시스템에는 과부하가 없음 (최대치 10점으로 제한)
  - 모든 에너지 획득은 최대치를 넘지 않도록 자동 조정

### 에너지 소모

- 특수 주사위 효과 발동
- 전략 카드 사용
- 방어막 생성

## 🃏 전략 카드 시스템

### 카드 획득

#### 카드 게이지 시스템 개요

**카드 게이지**는 전략 카드를 획득하기 위한 누적 포인트 시스템입니다.

- **초기값**: 0
- **최대값**: 5 (도달 시 카드 지급 후 0으로 초기화)
- **목적**: 플레이어의 행동과 성과에 따라 전략 카드 지급

#### 카드 게이지 획득 조건

**기본 조건** (라운드별 체크):

- 같은 눈금 2회 이상 → 카드 게이지 +1
- 연속 6 두 번 → 카드 즉시 지급 (게이지 소모 없음)
- 누적 점수 15점 달성 → 카드 게이지 +1

**성과 기반 조건** (연속성 체크):

- 2연패 시 → 카드 게이지 +1 (보상 시스템)
- 3연승 시 → 카드 게이지 +1 (성취 보너스)

#### 게이지 관리 규칙

**동시 조건 충족**:

- 한 라운드에서 여러 조건 충족 시 각각 독립적으로 적용
- 예: 같은 눈금 2회 + 15점 달성 → 게이지 +2

**게이지 상한선 및 초과 처리**:

- **상한선**: 게이지 최대값 5
- **상한선 도달 시**: 카드 1장 지급 후 게이지 0으로 즉시 초기화
- **상한선 초과 처리**: 게이지가 5를 초과할 경우 카드 지급 후 초과분은 다음 게이지로 이월
- **연속 처리**: 한 라운드에서 게이지가 5에 도달한 후 추가 조건 충족 시, 초과분이 새로운 게이지로 누적
  - 예: 게이지 4 상태에서 동시 조건 충족으로 +3 → 게이지 7이 되어 카드 지급 → 초과분 2가 새 게이지로 시작

**게이지 초기화**:

- 게이지 5 정확히 도달 → 카드 1장 지급 → 게이지 0으로 초기화
- 게이지 5 초과 → 카드 1장 지급 → 초과분이 새 게이지로 시작
- 즉시 지급 조건 (연속 6) → 게이지 변화 없음

### 카드 종류

#### 카드별 덱 구성 및 확률

- **공격 카드** (7장 - 35%): 상대 점수 감소 (에너지 2 소모)
- **방어 카드** (7장 - 35%): 다음 공격 무효화 (에너지 1 소모)
- **부스트 카드** (3장 - 15%): 이번 턴 점수 2배 (에너지 4 소모)
- **스틸 카드** (2장 - 10%): 상대 에너지 1~2점 훔치기 (에너지 2 소모)
- **리셋 카드** (1장 - 5%): 이번 라운드 점수 초기화 (에너지 4 소모)

### 카드 게이지 전략 시스템

#### 게이지 관리 전략

**적극적 게이지 수집**:

- 연승/연패 활용: 상황에 따라 승부 조절로 게이지 확보
- 점수 15점 타이밍: 중반 라운드에서 안정적 게이지 확보
- 다중 조건 노리기: 한 라운드에서 여러 조건을 동시에 충족하여 게이지 대량 획득

**게이지 보존 전략**:

- 게이지 2 상태 유지: 필요 시점까지 마지막 조건 회피
- 즉시 지급 활용: 연속 6으로 게이지 손실 없이 카드 확보

#### 게이지 타이밍 최적화

**초반 게임 (1-3라운드)**:

- 게이지 수집 우선: 카드 확보를 위한 기반 마련
- 안정적 조건 활용: 점수 15점 달성에 집중

**중반 게임 (4-6라운드)**:

- 전략적 활용: 축적된 게이지로 상황 대응
- 연승/연패 관리: 게임 흐름에 따른 게이지 확보

**후반 게임 (7라운드 이후)**:

- 즉시 활용 우선: 게이지보다 즉시 효과 중시
- 마지막 기회 활용: 연속 6 노리기

### 카드 사용 조건 시스템

- **긴급 상황**: 점수 차이 ±8 이상 → 즉시 사용
- **전략적 상황**: 에너지 품질이 높을 때 → 효과 극대화
- **방어 상황**: 상대가 공격 카드 보유 시 → 방어 카드 우선
- **공격 상황**: 상대 방어막 부족 시 → 공격 카드 집중

## 🛡️ 방어막 시스템

### 생성 및 관리

**생성 조건** (에너지 품질별):

- **일반 에너지 (1-3)**: 방어막 생성 불가
- **고급 에너지 (4-6)**: 에너지 3점으로 방어막 1개 생성
- **프리미엄 에너지 (7-10)**: 에너지 2점으로 방어막 1개 생성

**추가 제약**:

- **최대 동시 보유**: 2개까지
- **소모 방식**: 공격을 받을 때마다 방어막 1개씩 자동 소모되어 해당 공격 무효화
- **방어 보너스**: 방어 선언 성공 시 무료로 방어막 1개 추가 생성

### 효과

**차단 가능**:

- 공격 카드 (상대 점수 감소)
- ⚡ 공격 주사위 (점수 -2)
- 🔥 화염 주사위 (해당 라운드 점수 -3 부분만)
- 스틸 카드 (에너지 훔치기)

**차단 불가**:

- 🚫 차단 주사위 (모든 특수 효과 무효화)
- ⭐ 보너스 주사위 (자신에게 유리한 효과)
- 🎯 더블 주사위 (자신 점수 2배)
- 🔥 화염 주사위의 다음 턴 -1 효과
- 모든 지연 효과 (저주, 화염의 다음 턴 효과 등)
- 랜덤 이벤트 (폭탄, 점수 교환, 리셋 등)

## 🎭 랜덤 이벤트 시스템

### 발생 시점

- **발생 조건**: 모든 라운드 종료 직후, 무조건 이벤트 카드 덱에서 1장 드로우
- **덱 관리**: 덱 소진 시 셔플하여 재사용

### 덱 구성 (총 15장)

- **이벤트 없음**: 4장 (26.7%)
- **긍정적 이벤트**: 4장 (26.7%) - 행운의 여신, 점수 부스트, 보너스 턴, 에너지 충전
- **부정적 이벤트**: 4장 (26.7%) - 폭탄, 도둑, 저주, 에너지 소진
- **중립적 이벤트**: 3장 (20.0%) - 점수 교환, 리셋, 카드 교환

### 이벤트 종류

#### 긍정적 이벤트 (27% - 4장)

##### 🍀 행운의 여신

- **효과**: 주사위를 2번 굴려서 더 높은 값을 선택
- **적용 시점**: 다음 라운드 주사위 굴리기 직전
- **지속 시간**: 1라운드 (다음 라운드에만 적용)
- **방어막 차단**: 불가 (시스템 레벨 효과)
- **특이사항**: 주사위 결과를 선택할 수 있어 성향 선언과의 시너지 극대화 가능

##### ⚡ 점수 부스트

- **효과**: 이번 라운드 최종 점수에 +2 추가
- **적용 시점**: 라운드 종료 시 최종 점수 계산 직전
- **누적 규칙**: 다른 보너스와 중복 적용 가능
- **방어막 차단**: 불가 (자신에게 유리한 효과)
- **계산 순서**: 모든 다른 효과 적용 후 마지막에 +2

##### 🎲 보너스 턴

- **효과**: 이번 라운드에 추가로 주사위를 1번 더 굴림
- **적용 시점**: 정규 주사위 굴린 후 즉시 발동
- **점수 계산**: 두 주사위 결과를 모두 합산
- **성향 적용**: 두 번째 주사위도 선언한 성향 규칙 적용
- **특수 효과**: 두 번째 주사위에서도 특수 효과 발동 가능

##### 🔋 에너지 충전

- **효과**: 즉시 에너지 +3 획득
- **적용 시점**: 이벤트 발생 즉시
- **지급량 계산**:
  - 현재 에너지 0~7점: +3점 지급
  - 현재 에너지 8~9점: +2점 지급 (최대치 10점 초과 방지)
  - 현재 에너지 10점: +0점 지급 (이미 최대치)
- **과부하 처리**: 계산 오류로 최대치 초과 시 10점으로 조정
- **품질 분배**: 지급되는 에너지는 모두 일반 품질로 처리 (총 에너지양에 따라 품질 재계산)

- **적용 조건**: 현재 에너지가 3점 이상일 때만 발동

#### 카드 교환 상세 규칙

**발생 조건**:

- 중립적 이벤트 "카드 교환" 발생 시 (라운드 종료 후 10% 확률)
- 플레이어와 AI 모두 카드를 1장 이상 보유해야 교환 가능

**교환 방식**:

- 각자가 상대방의 카드 중 1장을 랜덤으로 선택하여 가져감
- 양쪽이 동시에 상대 카드를 선택하여 교환
- 교환 전 서로의 보유 카드 수만 공개 (카드 종류는 비공개)

### 이벤트 발동 타이밍 및 우선순위

#### 🕐 발동 타이밍 시퀀스

**라운드 종료 후 이벤트 처리 순서**:

1. **라운드 점수 확정**: 모든 주사위, 카드, 방어막 효과 적용 완료
2. **이벤트 카드 드로우**: 덱에서 1장 뽑기
3. **이벤트 발동 조건 확인**: 각 이벤트별 발동 조건 체크
4. **즉시 효과 적용**: 폭탄, 도둑, 점수교환, 리셋, 에너지충전 등
5. **지연 효과 예약**: 행운의 여신, 저주 등 다음 라운드 효과
6. **총점 합산**: 이벤트 효과까지 모두 반영한 최종 점수 계산

#### ⚡ 이벤트 우선순위 규칙

**동시 발생 시 처리 순서** (시스템적으로 동시 발생은 없지만, 효과 적용 순서):

1. **차단/무효화 효과** → 리셋 (모든 것을 초기화)
2. **감소 효과** → 폭탄, 도둑, 에너지소진, 저주
3. **교환 효과** → 점수교환, 카드교환
4. **증가 효과** → 점수부스트, 에너지충전, 보너스턴
5. **지연 효과 예약** → 행운의 여신, 저주

#### 🔄 이벤트 중복 처리

**같은 라운드 내 중복**:

- 불가능 (라운드당 1장만 드로우)

**연속 라운드 간 중복**:

- **저주**: 누적 가능 (저주 3개 = 다음 라운드 -6점)
- **행운의 여신**: 중복 불가 (새로운 효과가 기존 효과 덮어씀)
- **기타**: 각각 독립적으로 적용

#### 🎯 조건부 발동 규칙

**발동 조건 미충족 시**:

- **점수교환**: 점수 차이 3점 미만 → 효과 없음
- **도둑**: 대상 총점 5점 미만 → 효과 없음
- **에너지소진**: 현재 에너지 3점 미만 → 효과 없음
- **카드교환**: 한쪽이라도 카드 없음 → 효과 없음

## 📊 점수 시스템

### 점수 계산 체계 개요

**주사위 배틀**의 점수 시스템은 **라운드 점수**와 **총점**의 이중 구조로 운영됩니다.

- **라운드 점수**: 각 라운드에서 획득하는 임시 점수 (매 라운드 초기화)
- **총점**: 게임 전체에서 누적되는 최종 점수 (승부 결정 기준)

### 상세 점수 계산 순서

#### 1단계: 기본 점수 설정

**주사위 굴리기**:

- 플레이어와 AI가 각각 6면 주사위 1개 굴림
- 결과: 1~6 중 하나

**성향별 기본 점수 적용**:

- **공격 선언**: 4~6 성공(눈금 그대로), 1~3 실패(1점만)
- **방어 선언**: 2~5 성공(눈금 그대로), 1,6 실패(0점)
- **밸런스 선언**: 1~6 모두 성공(눈금 그대로)

#### 2단계: 상황 보너스 적용

**성향별 상황 보너스** (기본 점수가 성공 조건을 만족할 때만):

**공격 선언 상황 보너스**:

- 조건: 주사위 결과가 4~6 (성공) + 주사위 결과가 정확히 6
- 효과: 추가 +3점
- 예시: 주사위 6 → 기본 6점 + 상황 보너스 3점 = 9점

**방어 선언 상황 보너스**:

- 조건: 주사위 결과가 2~5 (성공)
- 효과: 방어막 +1개 (에너지 소모 없음)
- 예시: 주사위 3 → 기본 3점 + 방어막 1개 생성

**밸런스 선언 상황 보너스**:

- 조건: 항상 적용 (실패 없음)
- 효과: 에너지 +1, 카드 게이지 +1
- 예시: 주사위 4 → 기본 4점 + 에너지 1점 + 카드 게이지 1점

#### 3단계: 특수 주사위 효과 적용

**발동 조건과 효과**:

**공격 주사위 효과** (공격 선언 + 주사위 6 시 랜덤 발동):

- ⚡ **공격**: 상대 라운드 점수 -2
- 🔥 **화염**: 상대 라운드 점수 -3 + 다음 라운드 시작 시 -1
- 🎯 **더블**: 자신 라운드 점수 ×2

**방어 주사위 효과** (방어 선언 + 주사위 2 시 랜덤 발동):

- 🛡️ **방어**: 상대의 직접 공격 1회 무효화
- 🚫 **차단**: 이번 라운드 모든 특수 효과 무효화
- ⭐ **보너스**: 자신 라운드 점수 +2~6 (랜덤)

#### 4단계: 전략 카드 효과 적용

**카드 종류별 점수 영향**:

**공격 카드** (에너지 2 소모):

- 효과: 상대 라운드 점수 -3
- 적용 타이밍: 카드 사용 즉시
- 방어막 차단: 가능

**부스트 카드** (에너지 4 소모):

- 효과: 자신 라운드 점수 ×2
- 적용 타이밍: 카드 사용 즉시 (4단계에서 적용)
- 중복 규칙: 🎯 더블 주사위(3단계)와 중복 시 순차 적용하여 ×4 효과

**리셋 카드** (에너지 4 소모):

- 효과: 이번 라운드 점수를 0으로 초기화
- 대상: 양쪽 플레이어 모두
- 적용 타이밍: 카드 사용 즉시 (이후 효과들도 모두 무효)

#### 5단계: 방어막 판정

**방어막 작동 방식**:

- **자동 발동**: 공격을 받을 때 방어막 1개 자동 소모
- **차단 대상**: 직접적인 공격 효과 (⚡공격, 🔥화염 일부, 공격카드, 스틸카드)
- **차단 불가**: 🚫차단, ⭐보너스, 🎯더블, 지연 효과, 랜덤 이벤트

#### 6단계: 랜덤 이벤트 적용

**이벤트 카테고리별 점수 영향**:

**즉시 점수 변경 이벤트**:

- ⚡ **점수 부스트**: 라운드 점수 +2 (최종 단계에서 적용)
- 🧹 **리셋**: 라운드 점수 0으로 초기화
- 🦹 **도둑**: 총점에서 2점 이동 (대상 → 상대)

**총점 직접 조작 이벤트**:

- 💥 **폭탄**: 총점 ÷2 (소수점 버림, 최소 3점 보장)
- 🔄 **점수 교환**: 양쪽 총점 완전 교체

#### 7단계: 최종 점수 합산

**라운드 점수 → 총점 반영**:

1. 모든 효과가 적용된 라운드 점수 확정
2. 라운드 점수를 총점에 누적
3. 라운드 점수 초기화 (다음 라운드 준비)

### 점수 계산 상세 예시

#### 예시 1: 공격 선언 + 특수 효과 조합

**상황**: 공격 선언, 주사위 6, ⚡공격 효과 발동, 부스트 카드 사용

**계산 과정**:

1. 기본 점수: 6점 (공격 선언 성공)
2. 상황 보너스: +3점 (공격 선언 + 주사위 6)
3. 특수 효과: ⚡공격 → 상대 점수 -2
4. 카드 효과: 부스트 카드 → 자신 점수 ×2
5. **최종 결과**: (6 + 3) × 2 = 18점, 상대는 -2점

#### 예시 2: 방어막과 공격의 상호작용

**상황**: 상대가 🔥화염 효과 + 공격 카드 사용, 내가 방어막 2개 보유

**계산 과정**:

1. 🔥화염 효과: 내 점수 -3 예정
2. 방어막 판정: 방어막 1개 소모하여 🔥화염 차단 (이번 라운드 -3만, 다음 라운드 -1은 유지)
3. 공격 카드: 내 점수 -3 예정
4. 방어막 판정: 방어막 1개 소모하여 공격 카드 차단
5. **최종 결과**: 이번 라운드 피해 없음, 다음 라운드 시작 시 -1점, 방어막 0개

#### 예시 3: 이벤트와 카드의 충돌 처리

**상황**: 리셋 카드 사용 후 이벤트 🧹리셋 발생

**계산 과정**:

1. 리셋 카드 사용: 라운드 점수 0으로 초기화
2. 이벤트 🧹리셋: 이미 0이므로 추가 효과 없음
3. **우선순위**: 카드 효과가 먼저 적용되어 이벤트 효과 무의미
4. **최종 결과**: 양쪽 모두 이번 라운드 0점

### 시스템 간 점수 상호작용 규칙

#### 곱셈 효과 중복 규칙

**중복 가능한 곱셈 효과**:

- 🎯 더블 주사위 (×2) + 부스트 카드 (×2) = 최종 ×4
- 계산 순서: 기본 점수 확정 → 3단계에서 더블 적용 → 4단계에서 부스트 적용
- 예시: 기본 5점 → 더블로 10점 → 부스트로 20점

**중복 불가능한 곱셈**:

- 같은 종류의 카드는 라운드당 1장만 사용 가능
- 🎯 더블 주사위는 라운드당 최대 1회만 발동

#### 감점 효과 누적 규칙

**누적 가능한 감점**:

- ⚡공격 주사위 + 공격 카드 → 총 -5점
- 🔥화염 + 공격 카드 → 이번 라운드 -6점 + 다음 라운드 -1점
- 🦹도둑 + ⚡공격 → 총점 -2 + 라운드 점수 -2

**감점 방어 우선순위**:

1. 방어막으로 차단 가능한 효과 먼저 차단
2. 방어막 부족 시 피해가 큰 효과부터 차단
3. 지연 효과는 차단 불가능

#### 최솟값 보장 규칙

**라운드 점수 최솟값**:

- 일반적 최솟값: 0점 (음수 불가)
- 예외: 리셋 효과로 인한 0점은 다른 효과 적용 불가

**총점 최솟값**:

- 게임 진행 중: 1점 (0점 이하로 떨어지지 않음)
- 💥폭탄 이벤트: 최소 3점 보장
- 🦹도둑 이벤트: 최소 1점 보장

### 점수 기록 및 표시

#### 점수 구분 표시

**화면 표시 방식**:

- **라운드 점수**: 이번 라운드에서 획득한 임시 점수
- **총점**: 게임 시작부터 누적된 전체 점수
- **예상 총점**: 라운드 점수 반영 시 예상되는 총점

**점수 변화 애니메이션**:

- 증가: 녹색 상향 화살표 + 점수
- 감소: 빨간색 하향 화살표 + 점수
- 차단: 회색 X 표시 + 차단된 점수

#### 점수 히스토리 추적

**라운드별 기록**:

- 각 라운드의 최종 점수
- 적용된 효과 목록 (주사위, 카드, 이벤트)
- 방어막 사용 내역

**통계 정보**:

- 평균 라운드 점수
- 최고/최저 라운드 점수
- 특수 효과 발동 횟수
